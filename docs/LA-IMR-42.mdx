---

![](/img/IMR4-2-cutout.png)

Основные особенности:

* 4 IP канала (1080P) в реальном времени (120 к/с)
* Поддерживает один жёсткий диск объёмом до 5 Тб и SD карту объёмом до 512 Гб
* Встроенный конденсатор обеспечивает достаточно времени для завершения записи
* 60 минут записи при критически низком напряжении
* Форматы сжатия Н.264/H.265 c возможностью восстановления данных при экстренном отключении питания
